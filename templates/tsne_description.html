The dataset was collected from the The Whisky Exchange. Each whisky in the database has the following numerical attributes that contribute to its flavor profile:

  • Style: {Body, Richness, Smoke, Sweetness} on scale from 0-4
  • Binary indicators for 111 different flavors, where the top ten most common are: {Vanilla, Cinnamon, Smoke, Apple, Orange, Honey, Oak, Toffee, Lemon, Raisins}

Each whisky also has the following categorical attributes that can be used to support additional clustering, filtering, etc:

  • Country of Origin
  • Region
  • Age, yrs
  • Cask Type
  • Added Colouring
  • Proof

Thus, for each whisky we have a 115 dimensional feature vector describing the style, characteristics, and flavors in each dram. Obviously we cannot visualize 115 dimensions, but we can use statistical dimensionality reduction techniques such as t-SNE algorithm to condense our data down to 2-D for the sake of visualization. The algorithm tries to minimize the probability distribution differences between the full and reduced-order embeddings. For more information on the t-SNE algorithm and implementation check out this link.
